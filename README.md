# Своя игра Online — Render Perfect

Стабильная версия только под Render. Без App Store, Capacitor, PWA, service worker и лишних кэшей.

## Render settings

Build Command:

```bash
npm install
```

Start Command:

```bash
npm start
```

Environment Variables не нужны.

## Проверка после деплоя

Открой:

```text
https://svoya-online-game.onrender.com/api/health
https://svoya-online-game.onrender.com/api/rooms
```

`/api/health` должен вернуть `ok: true`.

После создания комнаты `/api/rooms` должен показать код активной комнаты.

## Важно про Render Free

На бесплатном Render сервер может засыпать. Если он заснул, активные комнаты исчезают, потому что комнаты хранятся в памяти Node.js процесса.

Для живой игры лучше:
- держать вкладку ведущего открытой;
- не ждать долго между созданием комнаты и входом игроков;
- для публичного продукта перейти на платный Render или VPS.
