# Своя игра Online

Онлайн-игра для двух команд. Ведущий создает комнату, игроки подключаются с телефонов.

## Локальный запуск

```bash
npm install
npm start
```

Открыть:

```text
http://localhost:3000
```

## Как играть

1. Ведущий нажимает "создать комнату".
2. Копирует ссылку и скидывает игрокам.
3. Игроки заходят с телефона, пишут имя и выбирают команду.
4. Ведущий открывает вопрос.
5. Игроки жмут кнопку "ЖМУ!".
6. Ведущий видит очередь нажатий и выбирает "верно" или "неверно".
7. Очки считаются автоматически.

## Онлайн-деплой

Подойдет любой Node.js-хостинг:
- Render
- Railway
- Amvera
- VPS

Команда запуска:

```bash
npm start
```

Порт берется из переменной `PORT`.
